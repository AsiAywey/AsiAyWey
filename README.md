====================================================================:

ms-auto: En Bootstrap 5, esta clase ("margin-start: auto") empuja todo el bloque de navegación a la derecha, dejando el logo solo a la izquierda.

transition: color 0.3s ease: No queremos que el rojo aparezca de golpe; esta línea hace que el color se "desvanezca" suavemente hacia el rojo al pasar el mouse, lo que da una sensación de mayor calidad.

!important: Lo usamos en los colores de los links porque Bootstrap a veces tiene reglas muy específicas que podrían ignorar tu CSS personalizado.

Flexbox (d-flex): Usado para alinear los iconos y el avatar perfectamente en el eje vertical (que no quede uno más arriba que otro).
